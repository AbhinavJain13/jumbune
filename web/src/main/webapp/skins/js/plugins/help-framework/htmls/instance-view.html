Below is the description of each field in the Instance view of debug analysis:

Parameters

  • Name
    • Identify the instance id of Mapper/Reducer of instrumented hadoop job that have run through the jumbune.
  • Input Keys
    • Identify the total no of input records of a specific instance of Mapper/Reducer.
  • Context Writes
    • Identify the total no of output records of specific instance of Mapper/Reducer.
  • Unmatched keys
    • Identify the total filtered keys of specific instance of mappers/reducer through the regex and user defined validation for a specific job.
  • Unmatched values
    • Identify the total filtered values of specific instance of mappers/reducer through the regex and user defined validation for a specific job.
