To create YAML file as per the cluster configurations, user can click on "Create YAML file" tab on the home page and start adding configurations in the wizard. User can enable or disable the Jumbune sections like if user only wants to run the profiling part user needs to enable only profiling, all other sections should be disabled. YAML file can be created on the basis of some parameters given in Create YAML wizard
