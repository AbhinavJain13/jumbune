Once all this steps gets completed now there are two options:

  • Save YAML File
    • To save YAML file for further use, click on save button in the wizard and then you can run the job by clicking on Run button.
  • Run YAML File
    • You can run YAML file directly also by clicking on Run button in the wizard.

Now here comes the analysis of your job for which you did all these basic steps. So Jumbune gives you the analysis of validating your hdfs input in form of HDFS validation, it gives you the cluster and JVM performance in form of Profiling, it gives you debug analysis in form of Debugger.
