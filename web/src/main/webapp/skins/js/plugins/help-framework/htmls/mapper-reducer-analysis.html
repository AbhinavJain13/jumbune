Below is the description of each field in the Mapper/Reducer view of debug analysis:

Parameters

  • Name
    • Identify the Mapper/Reducer class name of instrumented Hadoop job that have run through the Jumbune.
  • Input Keys
    • Identify the total no of input records of a specific Mapper/Reducer.
  • Context Writes
    • Identify the total no of output records of specific Mapper/Reducer.
  • Unmatched keys
    • Identify the total filtered keys of specific mappers/reducer through the Regex and user defined validation for a specific job.
  • Unmatched values
    • Identify the total filtered values of specific mappers/reducer through the Regex and user defined validation for a specific job.
