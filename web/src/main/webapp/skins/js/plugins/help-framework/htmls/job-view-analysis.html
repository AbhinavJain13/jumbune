Below is the description of each field in the Job view of debug analysis:

Parameters

  • Name
    • Identify the name of instrumented hadoop job that have run through the jumbune.
  • Input Keys
    • Identify the total no of input records and the value of this field must be same as total input records for all mappers.
  • Context Writes
    • Identify the total no of output records of specific instrumented job. The value of this field must be same as the total output records of all reducers.
  • Unmatched keys
    • Identify the total filtered keys of all mappers & reducer through the regex and user defined validation for a specific job.
  • Unmatched values
    • Identify the total filtered values of all mappers & reducer through the regex and user defined validation for a specific job.
