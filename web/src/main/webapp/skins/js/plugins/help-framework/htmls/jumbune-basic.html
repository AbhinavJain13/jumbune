Collects basic information about Hadoop Master (aliased for Namenode) and Slaves (the machines where Datanode and Tasktracker resides).

Parameters

  • Jumbune Job Name
    • Identify the name of Jumbune job. A folder would create with this specified name in the Jumbune_Home/jobjars folder and all information (logs/jar/counter files) would be stored in this folder.
Master Information
  • User
    • Username for the master machine.
  • Host
    • Host for the master machine.
  • RSA/DSA file
    • Path of the rsa/dsa file.
  • Agent Port
    • Port on which Jumbune agent has been started on hadoop master.
  • Jmx port on Master/Namenode
    • Identify the jmx port no of name node on Master machine.
  • Jmx Port on JobTracker
    • Identify the jmx port no of JobTracker on master machine.
Slave Information
  • Working Directory on Slaves
    • Directory on slave machine where temporary files related to Jumbune would be stored.
  • Jmx Port on DataNode
    • Identify the jmx port no of data nodes on slave machine.
  • Jmx Port on TaskTracker
    • Identify the jmx port no of TaskTracker on slave machine.
  • No of Unique Users
    • Number of different names of nodes in the cluster.
  • User
    • Username for the slave machine.
  • Host
    • Host for the slave machine.
  • Specify Range
    • If enabled, specify range of slave machines.Range must be containing full specified machine IP.
