Cluster profiler analysis gives analysis of the whole cluster based on some parameters. There are 3 sub sections in cluster profiler.

  • Data Center
    • This gives the racks of data centers and corresponding nodes in that rack. Jumbune also displays node ip of that node on hovering.
  • Categories: There are three different sub categories in Categories section in Jumbune.
    • I/O based stats.
    • JMX based stats that are inferred from hadoop Cluster.
    • JMX abstractionsn stats that ony Jumbune gives.

These sub categories contains further parameters so as to monitor Cluster Performance.

  • Network Latency
    • This gives the network traffic graph between all nodes in the cluster. The lighter is mesh line, lower is the network traffic and vice versa.

Also there is functionality of adding the parameters in favorites and trends. So as to monitor those parameters frequently.
