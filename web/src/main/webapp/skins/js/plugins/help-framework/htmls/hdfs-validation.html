Creates report which depicts HDFS data discrepancies. Validates HDFS data against data violation constraints.

Parameters

  • HDFS Input path
    • Identify the path of input data that user will need to validate.
  • Field Separator
    • Used to validate the no of fields in a record.
  • Record Separator
    • Used to validate the no of records in the input data.
  • Field Type
    • User can validate the fields with the specific field type.
  • RegEx
    • User can validate the input data with the reg ex comparison.
