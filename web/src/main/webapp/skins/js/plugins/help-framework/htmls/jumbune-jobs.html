Collects MapReduce Job Jar and Dependent Jar information.

Parameters

Main job jar information
  • Job jar location
    • Identify the location of input job jar file.
  • No of Jobs
    • Identify the total no of jobs in the input job jar file
  • Name
    • Identify name of the job.
  • Parameters
    • Identify the input parameters of input job.
  • Job Class Name
    • Identify the main class of input job.
